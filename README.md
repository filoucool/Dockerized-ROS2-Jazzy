## Prerequisites
- Docker installed on your system.
- A GitHub Personal Access Token with `read:packages`, `write:packages`, `delete:packages`, `repo` scopes stored in `TOKEN.txt`.

## Usage
**Build the Docker image:**
   ```bash
   ./build.sh
   ```
